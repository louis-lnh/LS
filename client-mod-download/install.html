Install Help

Add The Client Mod

Follow these steps after downloading the Lifesteal client mod. The mod is client-side, but it is required for the event features to display and work correctly.

01

Install Fabric

Open the official Fabric installer and install the loader for the Minecraft version required by the event. Use the normal Minecraft Launcher profile created by Fabric.

02

Download The Mod

Return to the download page and download shd-lifesteal-client-0.1.0.jar. Keep the file as a .jar; do not unzip or rename it.

03

Open Your Mods Folder

Put the JAR into your Minecraft mods folder. If the folder does not exist, create it inside your Minecraft directory.

  • Windows: %appdata%\.minecraft\mods
  • macOS: ~/Library/Application Support/minecraft/mods
  • Linux: ~/.minecraft/mods
04

Start Minecraft

Launch Minecraft with the Fabric profile, then join with the same Minecraft account that was approved for Lifesteal by SHD.

Before Joining

Quick Checks

  • The file is still named shd-lifesteal-client-0.1.0.jar.
  • The JAR is inside the mods folder, not the downloads folder.
  • You started the Fabric profile, not a vanilla Minecraft profile.
  • You are joining with the Minecraft account linked to your Discord/application.
Need Help?

Support

If Minecraft crashes, the server says you are missing the mod, or the custom event features do not appear, open a support ticket in Discord or use the SHD Support Portal.
